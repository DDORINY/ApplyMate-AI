# ApplyMate AI 서비스 범위

문서 기준 버전: `v0.4.0`

## 1. 서비스 정의

ApplyMate AI는 개인 사용자가 취업 준비 과정을 한 곳에서 관리할 수 있도록 돕는 AI 기반 취업 매니저다.

현재 구현된 범위는 커리어 프로필, 채용공고, AI 분석, 이력서, 지원 문서, 지원 현황 관리까지다. 일정 관리와 외부 캘린더/메일 연동은 이후 버전에서 단계적으로 확장한다.

## 3. 현재 제외 범위

- 외부 채용 사이트 자동 대량 크롤링
- 외부 채용 사이트 실제 자동 지원
- Google Calendar 실제 일정 생성
- Gmail 실제 메일 분석
- 유료 API 대량 호출
- 합격 가능성 확정 예측
- 다중 사용자 팀 협업
- 결제/구독

## 5. 외부 연동 원칙

- OpenAI 실제 호출은 `AI_PROVIDER=mock` 검증과 분리해 `NEEDS_VERIFICATION`으로 관리한다.
- Google Calendar 실제 일정 생성은 사용자의 명시적 승인 없이 수행하지 않는다.
- Gmail 실제 메일 읽기/분석은 사용자의 명시적 승인과 최소 권한 정책을 전제로 한다.
- 운영 Secret은 저장소에 커밋하지 않는다.

## 6. v0.4.0 기준 완료 판단

v0.4.0은 “채용공고 → 지원 문서 → 지원 현황” 흐름이 동작하는 상태를 완료 기준으로 한다. 사용자는 공고와 이력서, 지원 문서 버전을 선택해 지원 항목을 만들고 상태와 메모를 관리할 수 있다.

# 서비스 범위

ApplyMate AI는 개인 취업 준비 과정을 통합 관리하는 AI 취업 매니저를 목표로 합니다.

## MVP 포함 범위

- 이메일 기반 회원가입/로그인
- Google, GitHub 소셜 로그인 및 계정 연결
- 커리어 프로필 관리
- 이력/프로젝트/포트폴리오 관리
- 채용공고 등록과 분석
- 사용자-공고 적합도 분석
- AI 지원 문서 생성
- 지원 현황과 일정 관리
- 외부 캘린더/메일 연동

## MVP 제외 범위

- 결제 및 유료 플랜
- 관리자 페이지
- 모든 채용 사이트 대상 대규모 자동 크롤링
- 합격 가능성 보장 또는 예측
- 다중 사용자 협업 기능

## v0.1.3 완료 범위

- Google/GitHub OAuth provider 설정
- 소셜 로그인 시작 URL 발급
- OAuth callback 수신과 provider 사용자 정보 조회
- 1회용 login ticket 기반 프론트 토큰 교환
- OAuth provider token 미저장
- 동일 이메일 기존 계정 자동 병합 금지
- 로그인 후 명시적 소셜 계정 연결
- 연결된 소셜 계정 조회/해제
- 마지막 로그인 수단 해제 방지
- OAuth DB migration, API 문서, 환경변수 문서 업데이트

## v0.1.3 제외 범위

- Kakao, Naver, Apple 로그인
- Google Calendar/Gmail 연동
- GitHub repository 수집
- provider access token 저장 및 장기 동기화
- 운영 환경 OAuth 앱 생성 또는 production 배포

## 범위 관리 원칙

- 문서에 없는 기능은 선행 구현하지 않습니다.
- 새 기능은 버전 계획과 API/DB 문서에 먼저 반영합니다.
- 인증, 권한, 개인정보 보호 기준은 모든 사용자 데이터에 적용합니다.

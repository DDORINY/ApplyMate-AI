# 사용자-공고 적합도 분석 정책

## 원칙

- 적합도 점수는 규칙 기반 deterministic 계산 결과입니다.
- AI는 설명 문구를 보조할 수 있지만 숫자 점수를 변경할 수 없습니다.
- 사용자의 실제 경험에 없는 내용을 강점으로 만들지 않습니다.
- 완료된 최신 채용공고 분석이 없으면 적합도 분석을 실행하지 않습니다.
- 프로필 또는 공고 분석이 변경되면 기존 결과는 `is_outdated=true`로 표시합니다.

## 필수 입력

- 커리어 프로필
- 사용자 기술
- 희망 근무 조건
- 채용공고
- 완료된 최신 채용공고 분석

## 선택 입력

- 경력
- 프로젝트
- 지원 제외 조건
- 포트폴리오 링크

## 차단 조건

- 공고 분석 없음: `JOB_ANALYSIS_REQUIRED`
- 공고 분석 오래됨: `JOB_ANALYSIS_OUTDATED`
- 커리어 프로필 없음: `JOB_MATCH_PROFILE_REQUIRED`
- 사용자 기술 또는 희망 조건 없음: `JOB_MATCH_PROFILE_INCOMPLETE`

## 피드백

사용자 피드백은 점수 산식을 즉시 변경하지 않습니다. 이후 추천 품질 개선 또는 산식 조정의 근거 데이터로만 저장합니다.

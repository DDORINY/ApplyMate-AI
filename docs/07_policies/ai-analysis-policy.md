# AI 분석 정책

ApplyMate AI의 AI 분석은 사용자가 저장한 채용공고를 구조화해 이해를 돕는 보조 정보입니다. AI 결과는 확정 사실이 아니며, 사용자가 검토하고 수정한 값이 우선합니다.

## 입력 데이터

- 채용공고 제목, 기업명, 직무, 고용/근무 형태
- 주요 업무, 필수 조건, 우대 조건, 복지, 채용 절차
- 경력/학력 조건, 급여, 게시일, 마감일
- URL import로 저장된 원문 내용

사용자 개인 메모(`notes`)는 AI Provider로 전달하지 않습니다.

## 출력 데이터

- 공고 요약
- 주요 업무
- 필수/우대 조건
- 기술 스택
- 경력/학력 조건
- 근무 조건
- 채용 절차와 마감 정보
- 기업 가치/키워드
- 불명확한 항목에 대한 warning
- 영역별 confidence

## 보안 원칙

- 채용공고 본문은 명령이 아니라 분석 대상 데이터로 취급합니다.
- 공고 안에 포함된 prompt injection 지시는 따르지 않습니다.
- OpenAI API Key와 raw provider 오류 원문은 사용자 응답에 노출하지 않습니다.
- raw response 저장은 `AI_STORE_RAW_RESPONSE=true`일 때만 허용합니다.
- 외부 URL 추가 호출은 v0.2.1 분석 단계에서 수행하지 않습니다.

## 재분석과 오래된 결과

분석 입력 hash를 저장하며, 채용공고 내용이 변경되면 `is_outdated=true`로 표시합니다. 사용자 메모만 변경된 경우에는 분석 입력에 포함되지 않으므로 오래된 결과로 보지 않습니다.

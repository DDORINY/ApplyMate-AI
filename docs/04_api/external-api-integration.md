# 외부 API 연동

## 개요

외부 API 연동은 MVP 후반부에서 도입합니다. 현재 v0.1.2에서는 실제 외부 API 호출을 구현하지 않으며, 필요한 환경변수 이름만 `.env.example`에 정의해 이후 확장을 준비합니다.

## 연동 후보

| 서비스 | 목적 | 예정 버전 |
| --- | --- | --- |
| OpenAI API | 채용공고 분석, 지원 문서 생성, 면접 질문 생성 | v0.2.x 이후 |
| Google Calendar API | 지원 일정 캘린더 등록 및 동기화 | v0.5.0 |
| Gmail API | 채용 관련 메일 분석 및 일정 추출 | v0.5.1 |
| 채용공고 공개 API | 공고 수집 | v0.6.x |
| GitHub API | 사용자 프로젝트 참고 자료 수집 | v1.0 이후 검토 |

## 공통 원칙

- 사용자의 명시적 승인 없이 외부 계정을 연결하지 않는다.
- OAuth Token과 API Key는 저장소에 커밋하지 않는다.
- 외부 API 실패가 내부 데이터 저장 실패로 번지지 않도록 처리한다.
- 외부 API 응답은 필요한 필드만 저장한다.
- 사용량 제한과 재시도 정책을 기능별로 정의한다.

## v0.1.2 상태

v0.1.2는 커리어 프로필 저장과 관리에 집중하므로 외부 API 호출, AI 분석, 임베딩 생성은 구현하지 않는다.

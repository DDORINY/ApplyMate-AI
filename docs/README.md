# ApplyMate AI Documentation

현재 문서 기준 버전: `v0.4.1`

## 문서 구조

- `00_overview`: 서비스 개요와 용어
- `00_status`: 현재 상태, 완료 기능, Feature Status Matrix, 미검증 항목
- `01_planning`: 프로젝트 계획, 서비스 범위, 사용자 흐름
- `02_technical`: 기술 스택, 시스템 아키텍처, 기술 설계
- `03_requirements`: 기능/비기능/권한 요구사항
- `04_api`: API 명세, 오류 코드, 외부 API 연동
- `05_development-plan`: 버전 로드맵과 릴리스별 개발 계획
- `06_database`: DB 설계와 ERD
- `06_operations`: 배포, 환경변수, 테스트 체크리스트
- `07_policies`: 보안, AI, 채용공고, 지원 현황, 일정 정책
- `08_design`: 네비게이션 구조
- `09_guides`: 운영/외부 서비스 설정 가이드
- `10_ai`: AI 프롬프트와 출력 스키마
- `11_releases`: 버전별 릴리스 노트

# 채용공고 URL 등록 가이드

## 사용자 흐름

1. `/jobs/new` 화면에서 `URL 등록`을 선택한다.
2. 채용공고 URL을 입력한다.
3. 필요한 경우 기업명, 공고 제목, 본문을 직접 입력한다.
4. 시스템은 URL 안전성을 검증하고 HTML에서 제한적으로 내용을 추출한다.
5. 사용자가 입력한 값이 자동 추출값보다 우선 적용된다.
6. 등록 후 `/jobs/{jobId}` 상세 화면에서 내용을 검토하고 수정한다.

## 실패 시 처리

- URL이 사설망 또는 localhost를 가리키면 등록을 거부한다.
- HTML이 아니거나 응답이 너무 크면 등록을 거부한다.
- 제목/기업명을 자동 추출하지 못하면 기본값과 warning을 반환하고 상세 화면에서 수정하도록 한다.

## 운영 전 확인

- 대량 수집 기능을 추가하기 전 사이트 이용약관과 robots 정책을 검토한다.
- 사이트별 parser는 v0.2.1 이후 별도 설계 문서와 테스트를 추가한 뒤 구현한다.

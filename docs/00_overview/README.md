# ApplyMate AI 개요

ApplyMate AI는 개인용 AI 취업 매니저입니다. 사용자의 커리어 프로필과 채용공고 데이터를 연결해 공고 관리, 적합도 분석, 지원 문서 생성, 지원 현황/일정 관리를 단계적으로 제공하는 것을 목표로 합니다.

## 현재 버전

현재 구현 버전은 `v0.1.2`입니다.

완료된 범위:

- v0.1.0: 프로젝트 기반, Health API, Docker Compose 개발 환경
- v0.1.1: 회원가입, 로그인, JWT/Refresh Token 인증
- v0.1.2: 커리어 프로필 관리

## v0.1.2 핵심 기능

- 기본 커리어 프로필
- 기술 스택
- 경력
- 프로젝트
- 희망 조건
- 지원 제외 조건
- 포트폴리오 링크

## 다음 버전

v0.2.0에서는 채용공고 관리 기능을 구현합니다.

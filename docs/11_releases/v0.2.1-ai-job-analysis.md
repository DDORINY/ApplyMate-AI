# ApplyMate AI v0.2.1

## 1. 버전 개요

v0.2.1은 저장된 채용공고를 AI Provider로 구조화 분석하는 기능을 구현한 버전이다.

## 2. 개발 배경

채용공고 원문은 비정형 텍스트라서 적합도 분석과 지원 문서 생성에 바로 쓰기 어렵다. 공고의 업무, 조건, 기술, 경력, 마감 정보를 구조화해야 한다.

## 3. 해결하려는 문제

- 공고 본문 구조화 부재
- AI Provider 추상화 부재
- 분석 결과 저장/수정/삭제/재분석 부재
- 공고 변경 시 분석 최신성 판단 부재

## 4. 사용자 가치

사용자는 공고의 핵심 조건을 빠르게 확인하고, 이후 적합도 분석의 입력으로 사용할 수 있다.

## 5. 구현 범위

- 공고 본문 전처리
- input hash 생성
- `is_outdated` 판단
- AI Provider 추상화
- disabled/mock/openai Provider
- 구조화된 분석 결과 저장
- 분석 실행 이력 저장
- 분석 조회/수정/삭제/재분석
- 분석 상태 관리

## 6. 제외 범위

- 사용자-공고 적합도 계산
- 이력서/자기소개서 생성
- 지원 현황/일정 연동
- 자동 수집, pgvector, Celery

## 12. 보안 및 권한

- 모든 분석 API는 인증 필요

- provider secret은 응답/로그에 노출하지 않음
- raw response 저장은 설정값으로 제한
- prompt injection 텍스트는 데이터로 취급

# v0.7.0 AI 지원 문서 개선 루프 릴리스 노트

릴리스 기준일: 2026-07-20

## 요약

v0.7.0은 v0.3.3에서 만든 지원 문서 생성·버전 관리 기능 위에 AI 개선 루프를 추가한다. 사용자는 기존 문서 버전을 기준으로 개선안을 생성하고, 문장별 제안을 검토한 뒤 승인한 항목만 새 문서 버전으로 적용할 수 있다.

## Backend

- `document_improvement_runs`, `document_improvement_suggestions`, `document_improvement_sources`, `document_improvement_actions` 모델 추가
- Alembic migration `20260720_0100_create_document_improvement_tables.py` 추가
- 개선 실행 생성/목록/상세/재시도/삭제 API 추가
- 문장별 제안 승인·거절 API 추가
- 승인된 제안 적용 시 새 `application_document_versions` 생성
- 기준 버전보다 최신 버전이 있는 경우 적용 차단
- Mock AI provider에 구조화된 문서 개선 응답 추가

## Frontend

- 문서 상세에 `AI 문서 개선` CTA와 개선 이력 카드 추가
- `/documents/{documentId}/improve` 개선 요청 화면 추가
- `/documents/{documentId}/improvements/{runId}` before/after 비교와 문장별 승인 화면 추가

## API

- `POST /api/v1/documents/{documentId}/improvements`
- `GET /api/v1/documents/{documentId}/improvements`
- `GET /api/v1/documents/{documentId}/improvements/{runId}`
- `POST /api/v1/documents/{documentId}/improvements/{runId}/retry`
- `DELETE /api/v1/documents/{documentId}/improvements/{runId}`
- `PATCH /api/v1/documents/{documentId}/improvements/{runId}/suggestions/{suggestionId}`
- `POST /api/v1/documents/{documentId}/improvements/{runId}/apply`
- `POST /api/v1/documents/{documentId}/improvements/{runId}/reject`

# v1.0.0 MVP Release Notes

릴리스일: 2026-07-20  
대상 태그: `v1.0.0`

## 제품 목적

ApplyMate AI v1.0.0은 개인용 AI 취업 매니저의 MVP 릴리스다. 사용자는 커리어 프로필, 채용공고, 이력서, 지원 문서, 지원 현황, 일정, 추천, 알림을 하나의 흐름으로 관리할 수 있다.

## 포함 기능

- 회원가입, 로그인, 이메일 인증, 비밀번호 복구, 세션 관리
- 커리어 프로필, 경력, 프로젝트, 기술, 희망 조건 관리
- 채용공고 등록/수정/삭제, URL 기반 등록, 중복 검사
- mock provider 기준 AI 채용공고 분석, 적합도 분석, 이력서 분석
- 이력서 업로드, 다운로드, 텍스트 추출, 사용자 보정
- 근거 기반 지원 문서 생성과 AI 개선 루프
- 지원 현황과 일정 관리
- 대시보드 요약
- Google Calendar/Gmail 연동 기반 구조
- 규칙 기반 채용공고 추천, 추천 이력, 추천 설정
- in-app 알림, 알림 설정, due reminder worker
- request id, 보안 헤더, rate limit, health check, audit logs
- Docker Compose, E2E compose, Nginx reverse proxy 예시
- 사용자 가이드, 운영 가이드, 릴리스 체크리스트, 보안 리뷰

## 배포 절차 요약

1. 운영 DB backup을 수행한다.
2. 운영 환경변수를 저장소 밖에서 주입한다.
3. Docker image 또는 배포 artifact를 준비한다.
4. `alembic upgrade head`를 실행한다.
5. backend, frontend, worker를 시작한다.
6. `/api/v1/health/live`, `/api/v1/health/ready`, `/openapi.json` smoke를 확인한다.
7. 외부 provider는 운영 credential로 별도 live smoke를 수행한다.

## Rollback

문제가 발생하면 이전 이미지/커밋으로 되돌리고, DB 변경 영향이 있는 경우 backup restore 또는 검증된 Alembic downgrade 가능성을 확인한다. v1.0.0 자체는 신규 migration을 추가하지 않고 기존 head `20260720_0300`을 기준으로 릴리스된다.

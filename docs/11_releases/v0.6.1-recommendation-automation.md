# v0.6.1 Recommendation Automation Foundation

## Summary

v0.6.1 adds recommendation execution settings, run-if-due, snapshots, change detection, notification candidates, and recommendation UX improvements.

## Included

- User recommendation settings
- `MANUAL`, `DAILY`, `WEEKLY` execution frequency
- Authenticated `run-if-due` API
- Recommendation snapshot and snapshot item storage
- New/score up/score down/grade up/grade down/outdated change detection
- Recommendation notification candidates
- Recommendation list filters and change badges
- `/recommendations/history`
- `/settings/recommendations`
- Dashboard latest snapshot summary

## Excluded

- External job crawling
- AI/ML recommendation calls
- Real background worker operation
- Real email or push notification delivery
- Automatic application submission

# ApplyMate AI v0.2.0

## 1. 버전 개요

v0.2.0은 채용공고 관리 기능을 구현한 버전이다. 사용자는 직접 공고를 등록하거나 URL에서 제한적으로 HTML을 추출해 공고를 등록할 수 있다.

## 2. 개발 배경

커리어 프로필만으로는 취업 관리가 완성되지 않는다. 사용자가 관심 공고를 저장하고 상태를 관리할 수 있어야 AI 분석과 지원 관리로 확장할 수 있다.

## 3. 해결하려는 문제

- 채용공고 저장소 부재
- 기업 정보 재사용 구조 부재
- 사용자별 중복 공고 감지 부재
- URL 기반 등록 시 SSRF 방어 부재

## 4. 사용자 가치

사용자는 관심 채용공고를 저장하고 검색/필터/정렬하며, 이후 AI 분석 대상 데이터로 활용할 수 있다.

## 5. 구현 범위

- 기업 정보 저장 및 재사용
- 채용공고 직접 등록
- URL 기반 채용공고 등록
- 목록/상세/수정/삭제
- 검색, 필터, 정렬, 페이지네이션
- 상태, 관심 공고, 메모 관리
- 중복 공고 감지
- 사용자 소유권 검증
- SSRF 방어

## 6. 제외 범위

- AI 채용공고 분석
- 사용자-공고 적합도 분석
- 지원 현황 관리
- 일정/Google Calendar 연동
- 사이트별 scraper와 대규모 crawling

## 12. 보안 및 권한

- 모든 공고 API는 인증 필요
- `job_postings.user_id`로 소유권 검증
- URL 등록은 SSRF 방어 적용:
  - http/https만 허용
  - localhost/private/loopback/link-local/multicast/reserved IP 차단
  - DNS 검사
  - redirect 재검증
  - redirect 횟수 제한
  - timeout
  - HTML Content-Type 제한
  - 응답 크기 제한

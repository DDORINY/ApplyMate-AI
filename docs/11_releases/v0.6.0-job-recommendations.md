# v0.6.0 Rule-based Job Recommendations

## Summary

v0.6.0 adds rule-based recommendations for saved job postings.
추천 후보는 사용자가 저장한 채용공고로 제한하며, 추천 방식은 항상 `RULE_BASED`로 표시한다.

## Included

- Recommendation generation
- Recommendation list and detail
- Score, grade, reasons, missing requirements
- Blocking mismatch flag
- Outdated detection by profile/job/policy snapshot
- Feedback save/delete
- Recommendation run history
- `/recommendations` frontend page
- `/recommendations/{recommendationId}` detail page
- Dashboard top recommendation cards
- Job detail CTA to recommendation results

## Excluded

- External job crawling
- AI/ML recommendation
- Automatic application submission
- Automatic feedback learning
- Daily background scheduler

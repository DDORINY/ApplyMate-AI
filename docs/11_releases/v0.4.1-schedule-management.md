# v0.4.1 Schedule Management Release Notes

## Summary

v0.4.1 adds schedule management to ApplyMate AI. Users can manage application deadlines, coding tests, assignments, interviews, result announcements, and offer response deadlines linked to applications and job postings.

## Major changes

- Added `schedule_events`, `schedule_reminders`, and `schedule_event_history`
- Added `/api/v1/calendar` API group
- Added schedule create, list, detail, update, archive
- Added schedule status changes and history tracking
- Added schedule confidence
- Added in-app reminder persistence and UI display
- Added conflict detection and upcoming schedule lookup
- Added `/calendar`, `/calendar/new`, `/calendar/events/{eventId}` screens
- Added schedule entry points to application and job detail screens

## Exclusions

- Real email/push reminder delivery
- Real Google Calendar event creation
- Gmail schedule auto extraction
- Recurring schedules
- Automatic application status changes from schedules

# ApplyMate AI v0.1.2

## 1. 버전 개요

v0.1.2는 사용자 커리어 프로필 도메인을 구현한 버전이다. 이후 채용공고 적합도 분석과 지원 문서 생성의 근거 데이터가 되는 프로필, 기술, 경력, 프로젝트, 희망 조건, 제외 조건, 포트폴리오 링크가 추가되었다.

## 2. 개발 배경

AI 취업 매니저가 사용자에게 맞춤형 결과를 제공하려면 사용자의 실제 경력과 기술이 구조화되어 있어야 한다.

## 3. 해결하려는 문제

- 사용자 경력/기술 정보 저장 부재
- 프로젝트와 기술 연결 구조 부재
- 희망 조건과 제외 조건 부재
- 사용자 소유권 기반 프로필 API 부재

## 4. 사용자 가치

사용자는 자신의 커리어 정보를 등록하고 이후 채용공고 분석, 적합도 분석, 지원 문서 생성의 근거로 활용할 수 있다.

## 5. 구현 범위

- 커리어 프로필
- 사용자 기술
- 경력
- 프로젝트
- 프로젝트-기술 연결
- 희망 조건
- 지원 제외 조건
- 포트폴리오 링크
- `/profile` 화면

## 6. 제외 범위

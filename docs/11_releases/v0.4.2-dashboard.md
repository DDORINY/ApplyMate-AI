# v0.4.2 Dashboard

릴리스 목표: 지원 준비 현황을 한눈에 확인할 수 있는 읽기 전용 대시보드 제공

## 포함 범위

- `GET /api/v1/dashboard`
- 지원 상태 그룹별 집계
- 기간 필터와 사용자 지정 날짜 범위
- 오늘 일정, 이번 주 일정
- 다가오는 일정 마감과 마감 임박 공고
- 준비 중인 지원 항목
- 최근 공고 분석, 적합도 분석, 이력서 분석, 지원 문서
- 최근 활동 피드
- `/dashboard` 프론트 화면
- 헤더 주요 네비게이션에 대시보드 추가

## 제외 범위

- 대시보드 위젯 순서/표시 사용자 커스터마이징
- 차트 라이브러리 기반 고급 시각화
- 대시보드에서의 생성/수정/삭제 액션
- Google Calendar 실제 동기화

## API

- `GET /api/v1/dashboard`

Query:

- `period=7d|30d|90d|all|custom`
- `start_date`
- `end_date`
- `timezone`
- `recent_limit`

# ApplyMate AI v0.2.2

## 1. 버전 개요

v0.2.2는 사용자 커리어 프로필과 완료된 채용공고 분석 결과를 비교해 사용자-공고 적합도를 계산하는 버전이다.

## 2. 개발 배경

공고 분석만으로는 사용자가 지원 우선순위를 정하기 어렵다. 사용자의 실제 기술, 경력, 프로젝트, 희망 조건, 제외 조건을 기준으로 공고와의 적합도를 정량/정성 근거로 제시해야 한다.

## 3. 해결하려는 문제

- 사용자와 공고의 비교 결과 부재
- 기술/경력/프로젝트 근거 분리 부재
- 지원 제외 조건 반영 부재
- 적합도 결과 이력/피드백 부재
- 프로필 변경 후 결과 최신성 판단 부재

## 4. 사용자 가치

사용자는 관심 공고가 자신의 프로필과 얼마나 맞는지 확인하고, 부족한 기술과 위험 요소를 파악해 지원 여부를 결정할 수 있다.

## 5. 구현 범위

- 규칙 기반 적합도 점수
- 직무 적합도
- 기술 적합도
- 경력 적합도
- 프로젝트 적합도
- 근무 조건 적합도
- 위험/제외 조건
- 강점/보완점/위험 요소
- 추천 상태
- 실행 이력
- 피드백
- outdated 감지

## 6. 제외 범위

- AI가 숫자 점수를 생성/수정하는 기능
- 여러 공고 일괄 적합도 계산
- 추천 피드 자동 반영
- 지원서/자기소개서 생성

## 7. 시스템 변경사항

- `job_match` 라우터 추가

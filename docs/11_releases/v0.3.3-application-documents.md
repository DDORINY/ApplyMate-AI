# v0.3.3 Application Documents

## Summary

v0.3.3 adds grounded AI application-document generation. Users can create support documents connected to jobs, resumes, resume analysis, job analysis, and job match results; generate AI drafts; inspect evidence; edit and save versions; regenerate; restore; duplicate; and archive documents.

## Backend

- Added `/api/v1/documents` REST API.
- Added document generation, versioning, source snapshot, and generation run services.
- Added `GET /api/v1/ai/document-providers`.
- Extended AI provider protocol with `generate_application_document`.
- Added deterministic mock provider output with source references.

## Frontend

- Added `/documents`, `/documents/new`, `/documents/{documentId}`.
- Added main navigation item: `지원 문서`.
- Added textarea-based editor, live counts, version restore, evidence view, and generation run view.

## Database

# v0.3.0 이력서 파일 업로드

## 개발 배경

v0.2.2까지는 커리어 프로필과 채용공고 분석은 가능했지만, 사용자가 실제 이력서 파일을 저장하고 후속 AI 분석의 근거로 삼을 수 있는 기반이 없었다. v0.3.0은 이력서 파일 업로드와 관리 기능을 추가한다.

## 구현 목표

- PDF/DOCX 이력서 업로드
- 이력서 메타데이터 관리
- 파일 메타데이터 저장
- 파일 다운로드 및 삭제
- 사용자 소유권 검사
- 파일 업로드 보안 검증

## 사용자 가치

- 지원용 이력서를 서비스 안에서 관리할 수 있다.
- 기본 이력서를 지정해 후속 텍스트 추출/AI 분석 기능의 기준으로 사용할 수 있다.
- 파일명, 크기, MIME, 해시 등 추적 가능한 메타데이터가 저장된다.

## 구현 기능

- `resumes` 생성/목록/상세/수정/삭제
- 기본 이력서 지정
- `resume_files` 업로드/조회/다운로드/삭제
- LocalFileStorage adapter
- UUID 기반 내부 저장명
- SHA-256 파일 해시
- 사용자별 중복 파일 차단
- 확장자, MIME, 크기 검증

## 제외 기능

- OCR
- PDF/DOCX 텍스트 추출
- AI 이력서 분석
- S3 실제 저장소 연동
- 바이러스 스캔

## 보안

- 원본 파일명을 저장 경로로 사용하지 않는다.
- 내부 저장명은 UUID와 확장자 조합으로 생성한다.
- 업로드는 `.pdf`, `.docx`만 허용한다.
- MIME 타입을 허용 목록과 대조한다.
- 사용자별 파일 해시 중복을 차단한다.
- 다운로드와 삭제 전 사용자 소유권을 확인한다.

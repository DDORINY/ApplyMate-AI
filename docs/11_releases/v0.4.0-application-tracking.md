# v0.4.0 Application Tracking

## 요약

v0.4.0은 채용공고와 지원 문서를 실제 지원 현황으로 연결하는 버전이다. 사용자는 지원 상태, 지원 경로, 예정/완료일, 담당자, 메모, 상태 변경 이력을 관리할 수 있다.

## 주요 기능

- 지원 항목 생성/목록/상세/수정/보관
- 상태 변경 이력 자동 기록
- 지원 메모 관리
- 공고, 이력서, 이력서 파일, 지원 문서, 제출 문서 버전 연결
- 지원 문서 생성 화면의 숫자 ID 입력 제거 및 선택 UI 적용
- `/applications` 네비게이션 추가

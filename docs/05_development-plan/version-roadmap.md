## v0.1.0 프로젝트 기반 구축

### 목표

Frontend, Backend, Database의 기본 개발 환경을 구축한다.

### 개발 항목

* Next.js 프로젝트 생성
* FastAPI 프로젝트 생성
* PostgreSQL 연결
* Docker Compose 구성
* 환경변수 분리
* 공통 응답 구조
* 예외 처리
* Swagger 문서 구성

v0.1.0 첫 구현에서는 GitHub Actions는 제외하고 로컬 및 Docker 기반 개발 환경을 우선 구성한다.

### 완료 조건

* 전체 서비스 Docker 실행 성공
* Frontend에서 Backend Health API 호출 성공
* Database 연결 성공
* Redis 연결 상태 확인 가능

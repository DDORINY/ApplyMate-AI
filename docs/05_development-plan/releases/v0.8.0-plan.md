# v0.8.0 알림/리마인더 운영화 상세 개발계획

## 1. 목적

v0.8.0은 ApplyMate AI의 추천 알림 후보, 일정 리마인더, 지원 상태 변화 이벤트를 사용자에게 안전하게 전달하는 알림 운영 기반을 만든다.

## 2. 현재 기반

- v0.4.1 일정 reminder 저장
- v0.5.0 Google Calendar sync 기반
- v0.5.1 Gmail 후보 승인 기반 일정/상태 반영
- v0.6.1 추천 알림 후보 저장
- v0.7.0 지원 문서 개선 action 이력

## 3. 구현 범위 후보

- 알림 inbox API
- 알림 읽음/해제/보류 상태
- 일정 reminder due 조회
- 추천 알림 후보의 사용자 알림 전환
- 지원 상태 변경 알림 후보
- 사용자별 알림 설정
- mock sender 기반 이메일/푸시 발송 인터페이스
- 실제 발송 전 preview/approval 정책

## 4. 제외 범위

- 사용자 동의 없는 외부 메일/푸시 발송
- 운영 SMTP/푸시 서비스 실제 발송 완료 표시
- 자동 지원 제출
- 외부 채용 사이트 자동 수집

## 5. 안전 정책

- 알림은 사용자 소유 데이터만 사용한다.
- 민감 정보와 OAuth token은 알림 payload에 저장하지 않는다.
- 실제 발송은 사용자 설정과 운영 provider 검증 후에만 활성화한다.
- 실패한 외부 발송은 내부 알림 저장 실패로 이어지지 않게 한다.

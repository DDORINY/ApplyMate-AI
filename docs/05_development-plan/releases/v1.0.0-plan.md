# v1.0.0 MVP 릴리스 계획

## 1. 목적

v1.0.0은 ApplyMate AI의 MVP 릴리스다. v0.9.0까지 구현된 기능을 기준으로 인수 테스트, 운영 설정, 릴리스 문서, 사용자 가이드, 데모 데이터, 최종 보안 점검을 완료한다.

## 2. 기준 상태

- 최신 완료 버전: `v0.9.0`
- 기준 브랜치: `main`
- 작업 브랜치: `feature/v1.0.0-mvp-release`
- 현재 migration head: `20260720_0300`

## 3. 주요 범위

- 전체 MVP 인수 테스트
- 실제 배포 설정 점검
- 사용자 가이드 작성
- 운영 체크리스트 최종화
- 데모 데이터 준비
- 최종 보안/의존성 점검
- `NEEDS_VERIFICATION` 항목 재분류
- v1.0.0 릴리스 노트와 태그

## 4. 완료 기준

- Backend 전체 테스트 통과
- Frontend lint/type-check/build 통과
- E2E 인수 테스트 통과 또는 미검증 사유 기록
- 운영 환경변수 점검 완료
- DB migration chain 검증
- 배포/rollback 문서 확인
- Critical/High 보안 문제 없음
- v1.0.0 PR 병합 및 annotated tag push

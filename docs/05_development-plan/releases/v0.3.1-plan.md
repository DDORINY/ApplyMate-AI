# v0.3.1 이력서 텍스트 추출 계획

## 목표

v0.3.0에서 업로드한 PDF/DOCX 이력서 파일을 기반으로 텍스트를 추출하고, 이후 이력서 분석 및 AI 지원 문서 생성 기능에서 사용할 수 있는 근거 데이터를 저장한다.

## 구현 범위

- 업로드된 이력서 파일의 소유권 확인
- 저장소 파일 존재 여부 및 base directory 검증 재사용
- PDF/DOCX 텍스트 추출 실행 API
- 최신 추출 결과 조회 API
- 추출 결과 저장 테이블 추가
- 추출 성공/실패 상태, parser version, 원본 file hash snapshot 저장
- DOCX `word/document.xml`의 `w:t` 텍스트 추출
- PDF literal text operator 기반 best-effort 텍스트 추출

## 제외 범위

- OCR
- 이미지 기반 PDF 분석
- 이력서 항목 구조화 분석
- AI 기반 이력서 평가
- 커리어 프로필 자동 반영
- 외부 유료 파서 연동

## API

- `POST /api/v1/resumes/{resumeId}/files/{fileId}/extraction`
- `GET /api/v1/resumes/{resumeId}/files/{fileId}/extraction`

- Alembic migration 작성

- 사용자 소유권/미추출/성공 추출 테스트 작성
